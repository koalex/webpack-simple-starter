# webpack-simple-starter

#### Сначала нужно установить зависимости
```bash
npm install
```

##### Запуск сервера для разработки

```bash
npm run dev
```

##### Сборка для production

```bash
npm run build
```

##### Сборка для production и запуск статического сервера на порту :8080

```bash
npm start
```
